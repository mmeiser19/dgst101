WHAT IS JAVASCRIPT?

JavaScript is a programming language that allows websites to be interactive, "providing motion and behaviour to our limbs." This allows complexity to be added to webpages, rather than having them be static. This can "make your website appear to be more modern and sophisticated."

An example of JavaScript interactivity on a webpage would be a menu button "that can be clicked to display a selection of areas on the website that you can navigate to."

To the right is an image containing different menu icons that can be used with the help of JavaScript. The Hamburger menu is commonly-used, as is the Chocolate menu, whereas some of the others, such as the Oreos and Cake menus, are not used all that often.

An example of a well-known website/application that uses JavaScript is Google Maps, which allows users to zoom in or out of the map, along with being able to interact with businesses and locations that appear on the map.
