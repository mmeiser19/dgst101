WHAT IS HTML?

HyperText Markup Language(HTML) is what "defines the content and structure of your website."1 In other words, it is the skeleton of a webpage, providing the text and images that make up the content on a webpage. HTML also determines "what text and images show up and in what order"2 by the order they are written in the HTML file, along with what is included in said file.

HTML can create heading tags (<h1>, <h2>, etc.), which specify which type of header is used for certain text. Additionally, paragraph tags (<p>) can be used to indicate "body text," which is the text that makes up the body of the page, and which most of the text is written in.

4

Image tags (<img>) can be used to embed images in a webpage. Every img tag must have a source (src), which can either be a link or filepath of an image.

Image tags can also include alt text, denoted by the <alt> tag. Alt text is very useful for people who use screen readers, as it improves digital accessibility for those who cannot see or have difficulty with sight.


Hyperlink tags (<a href>) are used to specify a hyperlink to another webpage, either one that is part of your website or one that leads to an entirely different website. When using this type of tag, you must specify the URL of the page you want to link to.

Additionally, you can set other attributes to the (<a href>) tag, such as if the link will open in the same tab or in a new tab. One attribute that each (<a href>) tag must have is called anchor text, which is the clickable text that takes you to the new webpage.
