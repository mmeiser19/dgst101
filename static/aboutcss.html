WHAT IS CSS?

CSS creates the styling for a webpage, such as colors, layout, and formatting. Text can also be aligned(centered, to the left or right) Additionally, text can be given different fonts along with indentation through the use of CSS.

The stylization of webpages is done by "by adding 'style' code to the HTML code that’s already found on your page."1

When styling HTML tags, each style declaration has a property and a value. The property is a characteristic of an element and the value is a setting for the property. An example of a characteristic would be "color," and an example of a value for this property would be "red." In the image to the right, the property declaration states that text in <p> tags will be colored red.
